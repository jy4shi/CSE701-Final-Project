◆ gradient_descent()

int Input_Data::gradient_descent ( )

Public member function used to perform the gradient descent algorithm based on Input_Data, in order to improve efficiency, the step size is determined using the backtracking line search method.

Returns
int An integer that represents the case number
  • Case # 1: the current x point is now updated to a stationary point
  • Case # 2: reached the given maximum number of iterations, the current x point is updated, but it might not be a stationary point
  • Case # 3: the norm of current gradient is extremely large, algorithm is stopped to prevent overflow Gradient descent/ascent algorithm can be used to find either a local minimum or a local maximum, which is based on the input data that the user provided in input_function.txt

int Input_Data::newtons_method ( )

Public member function used to perform the newton's method algorithm based on Input_Data,.

Returns
int An integer that represents the case number
  • Case # 1: the current x point is now updated to a stationary point
  • Case # 2: reached the given maximum number of iterations, the current x point is updated, but it might not be a stationary point
  • Case # 3: the norm of current gradient is extremely large, algorithm is stopped to prevent overflow Due to the nature of the newton's method algorithm, it can only be used to find a stationary point near the initial point, that is, it cannot specifically find a local maximum or a local minimum
